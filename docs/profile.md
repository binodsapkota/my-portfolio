## IoT & connected systems

**Focus:** Gates, venues, warehouses, and retail floors—cards, readers, webcams, QR, events, vendor SDKs, and edge gateways unified with .NET and React.

### Systems delivered

#### 1. Access control & barrier

Multi-factor entry: QR codes, RFID and NFC card readers, and web cameras for capture or liveness—verified on a Raspberry Pi (or PC) at the edge, then barrier or security gate release when rules pass.

#### 2. People count

People counting at the entrance using the barrier and sensors so occupancy and flow stay accurate for safety, reporting, and capacity limits.

#### 3. Events & visitor auth

Event management tied to the gate: visitors authorized automatically on arrival from invite lists, tickets, or pre-registration—no manual check-in bottleneck.

#### 4. Hardware & vendor SDKs

Integration of webcams, attendance and biometric machines, RFID/NFC readers, and security / barrier gate controllers—wired into .NET and React for a single control plane.

#### 5. IoT gateways & warehouse assets

IoT gateway deployment for warehouse asset tracking: RFID tags, readers, and edge gateways report location and movement in real time so pallets, equipment, and inventory stay visible across the floor—synced to dashboards and the web stack.

#### 6. ESL e-paper — hardware & software

End-to-end electronic shelf label systems: e-paper tag hardware, wireless base stations or gateways, firmware for display refresh and low-power operation—plus software for price/promo sync from POS or ERP, template rendering, bulk updates, and store-floor health monitoring.
